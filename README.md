## **Getting Started**

### **Installation**

To get started with this project, clone the repository and install the dependencies:

```bash
https://github.com/Mirza2018/ronginbunon.git
cd ronginbunon
npm install
```

### **Running the App**
```bash
cd ronginbunon
npm run dev
```
